# 👨‍👩‍👧‍👦 مدرب التربية الوالدية الذكي

تطبيق ذكي يستخدم الذكاء الاصطناعي (OpenAI GPT-3.5) لتقديم مشورة والدية متخصصة وعملية.

## ✨ المميزات الرئيسية

✅ **واجهة عربية متكاملة** - RTL كامل مع ألوان احترافية  
✅ **تحليل شامل** - تقسيم الردود إلى 4 أقسام منظمة  
✅ **خطط قابلة للتطبيق** - 3 خطوات أسبوعية محددة وقابلة للقياس  
✅ **سيناريوهات حوارية** - نصوص مقترحة جاهزة للاستخدام  
✅ **أمان عالي** - تحذيرات عند الحاجة لاستشارة متخصصين  
✅ **سهولة الاستخدام** - نموذج بسيط للإدخال  

---

## 🚀 البدء السريع

### 1. استنساخ المستودع
```bash
git clone https://github.com/dykydoyd-YASMEEN/Helper-parenting.git
cd Helper-parenting
```

### 2. تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### 3. إعداد OpenAI API Key
```bash
# انسخ الملف
cp .env.example .env

# أضف مفتاح API خاصك
# قم بتحرير .env وأضف:
OPENAI_API_KEY=sk-your-key-here
```

[احصل على مفتاح API من OpenAI](https://platform.openai.com/api-keys)

### 4. تشغيل التطبيق
```bash
streamlit run streamlit_app.py
```

سيفتح التطبيق في متصفحك على: `http://localhost:8501`

```
📌 القسم 1: بيانات المراهق
├─ اسم المراهق (مطلوب) *

├─ وصف المشكلة (مطلوب) *

📌 القسم 3: السياق والبيئة
├─ مستوى التوتر في المنزل *
└─ المحاولات السابقة (اختياري)

📌 القسم 4: معلومات إضافية
└─ اهتمامات المراهق (اختياري)
```

اضغط على زر **"✅ حفظ جميع البيانات والبدء"** للانتقال لمنطقة المحادثة.

### الخطوة 3: المحادثة

- ستظهر بطاقات المعلومات في الأعلى
- اكتب سؤالك في مربع الإدخال
- اضغط Enter أو زر الإرسال
- المدرب سيرد بـ 4 أقسام:
  1. **📌 تحليل الموقف** - فهم السلوك
  2. **📋 خطة الأسبوع** - 3 خطوات عملية
  3. **💬 سيناريو الحوار** - نص مقترح
  4. **⚠️ ما يجب تجنبه** - الأخطاء الشائعة
